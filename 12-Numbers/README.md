✔️ spread operator(스프레드 연산자)
- 배열, 문자열, 객체 등 반복 가능한 객체를 개별 요소로 분리할 수 있다.
```js
let arr1 = [1,2,3,4,5]
let arr2 = [...arr1, 6,7,8,9];
console.log(arr2) //(9) [1, 2, 3, 4, 5, 6, 7, 8, 9]

var str1 = 'sudal'; 
var str2 = [...str1]; 
console.log(str2); //(5) ['s', 'u', 'd', 'a', 'l']
```
